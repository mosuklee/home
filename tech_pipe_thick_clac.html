PIPE THICKNESS CALCULATION

1. 배관의 두께 계산식

설계온도 및 설계압력하에서 최소 관벽 두께는 ASME Code 기준에 의하여 아래와 같은 공식으로 계산한다.


tm = [ (P x Do) / (2 x (SE + Py)) ] + A 


여기에서
tm = 최소관벽 두께 (mm)
P : 설계압력 (kg/cm2.g)
Do : 관 외경 (mm)
S : 설계온도에서의 최대 허용응력 (kg/cm2.g)
E : 용접효율 (ASME Section 8 : SMLS의 경우 1로 적용)
y : 계수(ANSI/SAME B31.1 Table 104.1.2(A))
A : 부식여유 (ASME B31G : 1.6mm)
