It is recommended the the capacity of the venting equipment be not less than the values shown in Table 9 at the design suction pressure to insure adequate removal capaicty under commercial operating conditions.

6.5.1 Peocedure for Siazing Venting Equipment. (Venting Equipment의 크기 선정과정)

6.5.1.1 Determine total steam flow of the unit by adding the main turbine exhaust flow and any auxiliary turbine exhaust flow entering all shells of the condenser. ( Main turbine exhaust 유량과 컨덴서로 들어가는 auxiliary turbine exhaust유량을 더한 값을 Total steam 유량으로 결정합니다. )

6.5.1.2 Determine the total number of MAIN turbine exhaust opening of all shells. Do not include auxiliary turbine exhaust openings. ( 모든 Shell의 MAIN turbine 배기구의 전체 수를 결정합니다. 보조터빈의 배기구는 포함하지 않습니다. )

6.5.1.3 Divide flow obtained in 6.5.1.1 by exhaust opening number obtained in 6.5.1.2. The resultant number is the EFFECTIVE STEAM FLOW EACH MAIN EXHAUST OPENING. ( 6.5.1에서 구한 유량을 6.5.1.2에서 구한 배기구 수로 나눕니다. )

6.5.1.4 Enter the appropriate section of Table 9 based on whether unit is a single shell, twin sell or triple shell condenser and locate the flow obtained in Step 6.5.1.3 tin the left vertical culumn. ( 장치가 단일 쉘, 트윈쉘 또는 트리플 쉘 콘덴서인지에 따라 표9의 해당 섹션을 찾고 왼쪽 수직 기둥에서 6.5.1.3 단계에서 얻은 유량값의 해당부분을 찾습니다. )

6.5.1.5 Determine TOTAL NUMBER OF EXHAUST OPENINGS for all shells by adding the total number of main trubine exhaust openings to the total number of auxiliary turbines exhausting in the ondenser ( 콘덴서에서 배기되는 Auxilary turbine exhausting의 총수에 Main turbine exhausting의 총수를 더하여 "TOTAL NUMBER OF EXHAUST OPENINGS" 를 결정합니다. )

Split auxiliary turbine exhaust ducts coming from one auxilary turbine count as one auxiliary turbine exhaust. ( 하나의 Auxilary tuebine exhaust로부터 나오는 분할 Auxilary turbine exhaust ducts는 하나의 Auxiliary tuebine exhaust로 계산됩니다. )

6.5.1.6 Locate the oppropriate column and capacity using the number obtained in 6.5.1.5 ( 6.5.1.5에서 얻은 숫자를 사용하여 적정한 칼럼과 용량을 찾습니다. )

6.5.1.7 If inderpendent venting system are utilized for each shell of a multi-shell condenser, the capacity of each system is determined by dividing the total capacith obtained from the appropriate table by the number of indenpendent venting system.
( Multy-shell condenser의 각 쉘에 대하여 독립 Venting system을 구성하는 경우, 각 시스템의 용량은 해당 적정한 표로부터 얻은 총 용량을 독립 venting system으로 나누어 결정합니다. )

6.6 Rapid Evacuation Equipment

When starting a turbine, it is desirable to reduce the condenser pressure from atomospheric to some lower value. This can be done by means of single stage ejector or is dependent on the effectiveness of the turbine gland seals, the volume of the condenser shells, turbine casings, and associated ducting as well as the time desired for such reduction. Where specific values are not listed, refer to Table 8.
