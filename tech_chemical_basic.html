화학공학 상식

분율 (fraction)

wt% 질량유량에 의한 %를 의미함
vol% 체적유량에 의한 %를 의미함
mol% mol 분율에 의한 %를 의미함
Note) vapor fraction : 전체량중 기체의 분율 (증기에서는 건도)


체적유량을 질량유량으로 환산하는 방법

분자량 1mol에 포함된 물질의 질량(g)
체적 1mol에 포함된 물질의 체적 (22.415m3/kmol) (Standard state : at 0℃, 1atm)
수식

kg/h = 체적(m3/h) $\frac{분자량(kg)}{1(kmol)}$ x $\frac{1(kmol)}{22.43(m3))}$ x $\frac{Act.Pres(kg/cm2.a)}{1.03323(kg/cn2.a))}$ x $\frac{273.15(K)}{273.15+Act.Temp(℃))}$
