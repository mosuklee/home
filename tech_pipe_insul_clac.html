배관 열손실 계산 (Pipe Heat Loss Calculation)


1. 배관의 열손실 계산식

원형 배관에서의 열손실 계산은 다음수식에 의하여 계산된다.


QL = { 2 x 𝜋 x (Ti - To) } / Σ𝑅 

여기에서
QL : Heat Loss (배관의 단위길이당 열손실 (kcal/h.m))
Ti  : 배관의 내부 유체온도 (℃)
To : 배관의 외부 유체온도 (℃)
Σ𝑅 : 배관의 열저항(m.h.℃ / kcal)


Σ𝑅 = Ri + R1 + R2 + R3 + R4 + R5 + .... + Ro 

여기에서
Ri  : 1 / ( 𝛼i x r1 )      ... Film
R1 : LN(R2/R1) / 𝜆1   ... Steel
R2 : LN(R3/R2) / 𝜆2   ... Insulation
R3 : LN(R4/R3) / 𝜆3   ... Air Gap
R4 : LN(R5/R4) / 𝜆4   ... Steel
Ro : 1 / ( 𝛼o x r5 )     ... Film

2. 배관의 표면 온도 계산

원형 배관에서의 표면온도 계산은 다음수식에 의하여 계산된다.


tw = ti - (Σ𝑅(x) / 𝑅) x * ( ti - to)

여기에서
tw   : 배관 벽면에서의 온도 (℃)
ti   : 배관의 내부 온도 (℃)
To   : 배관의 외부 온도 (℃)
Σ𝑅(x) : 배관의 x Layer까지의 열저항(m.h.℃ / kcal)
𝑅   : 배관의 전체 열저항 (m.h.℃ / kcal)
