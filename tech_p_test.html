내압검사 기준 (Hydrostatic/Pneumatic Test)


배관의 내압검사 기준


적용코드

     ASME B31.1/31.3
     - Hydrostatic Test 기준 : Section 137.5.1 / Paragraph 345.4.2
     - Pneumatic Test 기준 : Section 137.5.2 / Paragraph 345.5

시험방법

[수압시험]
구 분 압 력 시 간 비 고
시험압력 ≥ 1.5 × 설계압력 ≥ 10분 ≥ 압력유지 확인
누설시험 ≥ 1.0 × 설계압력 ≥ 10분 ≥ 육안검사 수행

[기압시험]
구 분 압 력 시 간 비 고
시험압력 ≥ 1.1~1.25 × 설계압력
(최대 1.5배 제한)		 ≥ 10분 ≥ 압력유지 확인
누설시험 ≥ 1.0 × 설계압력 ≥ 10분 ≥ 육안검사 수행



압력용기 내압검사 기준


적용코드

     ASME BPVC
     - Hydrostatic Test 기준 : Sec VIII Div.1/UG-99
     - Pneumatic Test 기준 : Sec VIII Div.1/UG-100

시험방법

[수압시험]
구 분 압 력 시 간 비 고
시험압력 ≥ 1.3 × MAWP (최대허용작동압력)
또는 보정식 적용		 ≥ 10분 ≥ 압력유지 확인
누설시험 ≥ 1.0 x MAWP (최대허용작동압력) ≥ 10분 ≥ 육안검사 수행

[기압시험]
구 분 압 력 시 간 비 고
시험압력 ≥ 1.1 × MAWP,
≤ 1.15 × MAWP		 ≥ 10분 ≥ 압력유지 확인
누설시험 ≥ 1.0 x MAWP ≥ 10분 ≥ 육안검사 수행



내압검사 보고서

내압검사 보고서는 다음과 같은 서류로 구비되어야 한다.
     작성서류
     - 내압검사 부위 표시도면
     - 내압검사 보고서
     - 내압검사 사진대지
